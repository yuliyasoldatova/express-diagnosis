# Что сейчас забирает вашу энергию? — Мини-диагностика

Статический сайт-диагностика состояния для женщин. Случайно показывает одну из 8 карт с описанием сферы, которая может забирать энергию.

---

## Файлы проекта

```
index.html   — разметка (три экрана: приветствие, загрузка, результат)
style.css    — все стили и анимации
script.js    — массив карт и логика рандома
README.md    — эта инструкция
```

---

## 1. Как загрузить проект на GitHub

1. Зайдите на [github.com](https://github.com) и войдите в аккаунт (или создайте).
2. Нажмите кнопку **New** (зелёная кнопка) → создайте новый репозиторий.
3. Назовите его, например: `energy-diagnosis` (только латиница, без пробелов).
4. Оставьте репозиторий **Public** (публичный) — это нужно для GitHub Pages.
5. Нажмите **Create repository**.
6. Загрузите файлы: нажмите **Add file → Upload files** и перетащите все четыре файла проекта.
7. Нажмите **Commit changes**.

---

## 2. Как включить GitHub Pages

1. Перейдите в **Settings** вашего репозитория (вкладка вверху).
2. В левом меню найдите раздел **Pages**.
3. В разделе **Source** выберите ветку **main** (или **master**) и папку **/ (root)**.
4. Нажмите **Save**.
5. Через 1–2 минуты сайт будет доступен по адресу:  
   `https://ВАШ_ЛОГИН.github.io/energy-diagnosis/`

## 3. Где заменить Telegram-ссылку

Откройте файл `index.html` и найдите строку:

```html
<a class="btn-primary" href="ВСТАВИТЬ МОЮ ССЫЛКУ НА TELEGRAM" target="_blank" rel="noopener">
```

Замените `ВСТАВИТЬ МОЮ ССЫЛКУ НА TELEGRAM` на вашу реальную ссылку, например:

```html
<a class="btn-primary" href="https://t.me/ваш_username" target="_blank" rel="noopener">
```

## 4. Где редактировать тексты карт

Все тексты хранятся в файле `script.js` в массиве `cards`.

Каждая карта имеет поля:
- `theme` — тема карты (отображается мелким шрифтом сверху)
- `title` — название карты (крупный заголовок)
- `subtitle` — подзаголовок курсивом
- `energyDrain` — раздел «Что забирает энергию»
- `interpretation` — раздел «Расшифровка»
- `action` — раздел «Шаг на ближайшие 24 часа»

Просто отредактируйте нужный текст в кавычках.

## 5. Где менять цвета и стили

Откройте файл `style.css`. В самом начале файла есть блок `:root` с переменными цветов:

```css
:root {
  --cream: #faf7f2;        /* основной фон */
  --rose: #d9a8a0;         /* розовый акцент */
  --gold: #c9a96e;         /* золотой акцент */
  --beige: #e8ddd0;        /* бежевый */
  --graphite: #2c2825;     /* основной цвет текста */
  /* ... */
}
```

Замените значения HEX-цветов на нужные вам.

Для шрифтов в начале `style.css` есть переменные:

```css
--font-display: 'Cormorant Garamond', Georgia, serif; /* заголовки */
--font-body:    'Jost', sans-serif;                   /* основной текст */
```

Шрифты подключены из Google Fonts. Если хотите другой шрифт — замените название в переменной и обновите ссылку `<link>` в `index.html`.
